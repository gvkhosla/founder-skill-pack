# focus-planner

## Invocation
- Help me focus
- Plan next week's focus

## Outputs
- focus-plan.md

## Prompt
# Focus Planner

Produce `focus-plan.md` with:
1. One primary focus
2. What gets deprioritized
3. Why this focus matters now
4. What success looks like in one week

Rules:
- choose one focus, not a bundle
- make tradeoffs explicit
