How it works
Invoke. Framework. Artifact.

Every skill follows the same pattern — structured questions, a proven framework, one file. Nothing else.

💬

Say the phrase

Each skill has a plain-English trigger. "Scope my MVP." "Map my assumptions." "Diagnose my churn." Your agent knows what to do next.

⚙️

The framework runs

The skill asks the right questions and applies the right expert framework — Jobs To Be Done, Mom Test, Hook Canvas, April Dunford's positioning method.

📄

You get a file

Not advice. Not a framework overview. A concrete .md artifact — specific decisions you can act on, share, argue with, and build from.

Reads everything
before it speaks.

The only proactive skill. It reads your full product history, then tells you what it sees — before you ask anything. Use it when you don't know where to start.

Install
Legacy pack + OS beta.

The published pack still installs with npx founder-skills. The new Founder Skills OS beta adds generated coding-host bundles and install/export commands for local repo workflows.

# Published legacy pack
npx founder-skills install --agent pi
npx founder-skills install --agent claude
npx founder-skills install --agent codex --out ./AGENTS.founder-skills.md
This is the existing package install path for the original 27-skill pack.
# Founder Skills OS beta for pi
npm install
npm run os:gen
npm run os:install -- --host pi
Exports to ~/.pi/agent/skills/founder-skills-os/.
# Founder Skills OS beta for Claude Code
npm install
npm run os:gen
npm run os:install -- --host claude-code --scope project
Exports to .claude/skills/founder-skills-os/ and updates a managed section in CLAUDE.md.
# Founder Skills OS beta for Codex
npm install
npm run os:gen
npm run os:install -- --host codex
Exports to .codex/founder-skills-os/ and updates a managed section in AGENTS.md.
# Founder Skills OS beta for OpenCode
npm install
npm run os:gen
npm run os:install -- --host opencode
Exports to .opencode/founder-skills-os/ and updates a managed section in AGENTS.md.
# Founder Skills OS beta for OpenClaw
npm install
npm run os:gen
npm run os:install -- --host openclaw
Exports to .openclaw/founder-skills-os/ and updates a managed section in AGENTS.md with lite/full OpenClaw prompts.

Exports to ~/.hermes/skills/founder-skills-os/. Verify with hermes skills list.
